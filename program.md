# Dawnop-site

1. 我需要写一个博客性质的个人网站，整体的风格要简单，干净。

   1. 博客前端包括首页，文章页，关于页。文章页要支持markdown和和内嵌latex的显示。后台要求支持管理文章，markdown上传下载。
   2. 另一个就是后台的文件的上传和下载功能，对于图片和文本要支持预览，使用第三方对象存储api，文档见 https://developer.qiniu.com/kodo/3939/overview-of-the-api。
   3. 后台需要登陆才能进入。

2. 技术路线上要求尽可能使用python后端，因为比较简单，通过前后端分离的方式开发，因为后续的网盘可能要支持其他客户端。数据库SQLite就够用了。

3. 记得创建git项目，并且coauther不要带上Claude。记得以开源为标准，一些敏感信息要放入配置文件中，不commit。

4. 代码在本地开发&本地测试，无误后再到服务器上测试。

5. 我需要在4核4G机器上运行该网站，登陆地址是 ssh <user>@<server>（真实地址不入库）。其中80端口是打开的，如果要测试结果的话，可以使用公网dawnop.com来测试。
